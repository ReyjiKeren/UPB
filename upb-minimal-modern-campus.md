\# UNIVERSITAS PELITA BANGSA

\## Minimal Modern Campus Website

\### Antigravity Vibe Code Documentation



---



\## 1. TUJUAN WEBSITE

Website resmi Universitas Pelita Bangsa dengan konsep:

\- Minimalis

\- Modern

\- Profesional

\- Cepat diakses di semua device (mobile, tablet, desktop)



Fokus utama:

\- Memberikan informasi kampus secara jelas

\- Menarik calon mahasiswa

\- Meningkatkan kepercayaan pengguna



Tanpa backend (static website).



---

\## 3. KONSEP DESAIN UI / UX



\### 3.1 Gaya Desain

\- Minimal Modern

\- Clean layout

\- Banyak white space

\- Fokus pada keterbacaan



---



\### 3.2 Warna (Brand Identity)

Mengikuti warna logo Universitas Pelita Bangsa:

\- Primary Blue: #0B3C88

\- Secondary Blue: #1F6FDB

\- Accent Red (CTA): #E53935

\- Background: #F9FAFB

\- Text Utama: #1F2937

\- Border: #E5E7EB



---



\### 3.3 Tipografi

\- Heading: Poppins (SemiBold / Bold)

\- Body Text: Inter (Regular / Medium)

\- Button \& Label: Inter Medium



---



\### 3.4 Responsive Design

\- Mobile First

\- Grid fleksibel

\- Konten maksimal width 1200px (desktop)

\- Navigasi hamburger untuk mobile



---



\### 3.5 Navigasi

\- Navbar sticky

\- Scroll halus antar section

\- CTA “Daftar” selalu terlihat



---



\## 4. TEKNOLOGI YANG DIGUNAKAN



\### 4.1 Frontend

\- HTML5 (semantic structure)

\- CSS3 (Flexbox \& Grid)

\- JavaScript ES6



---



\### 4.2 Framework \& Tools (Opsional)

\- Tailwind CSS (utility-first)

\- Google Fonts (Poppins \& Inter)

\- Icon SVG (Heroicons / Feather)



---



\### 4.3 Performance

\- Tanpa library berat

\- Tanpa video autoplay

\- Gambar terkompresi

\- Animasi ringan berbasis CSS



---



\## 5. FITUR INTERAKTIF (TANPA BACKEND)



\### 5.1 Interaksi UI

\- Hover effect pada card

\- Button ripple effect

\- Smooth scroll navigation

\- Sticky navbar dengan shadow dinamis



---



\### 5.2 Animasi

\- Fade-in on scroll

\- Slide-up ringan

\- Count-up animation pada statistik



---



\### 5.3 Konten Dinamis (Client-side)

\- Filter Program Studi

\- Expand / collapse detail fakultas

\- Tab konten tanpa reload halaman



---



\### 5.4 Aksesibilitas

\- Kontras warna aman

\- Font mudah dibaca

\- Navigasi keyboard-friendly



---



\## 6. PRINSIP ANTIGRAVITY VIBE

\- Struktur jelas dan konsisten

\- Komponen modular

\- UI bersih tanpa distraksi

\- Fokus pengalaman pengguna

\- Estetika modern \& profesional



---



\## 7. CATATAN PENTING

\- Website bersifat static

\- Tidak menggunakan database

\- Tidak menggunakan server-side processing

\- Siap di-host di static hosting



---



\## 8. HASIL YANG DIHARAPKAN

\- Website kampus modern

\- Mudah dipahami

\- Cepat diakses

\- Menarik calon mahasiswa

\- Siap dipresentasikan menggunakan Google Antigravity
